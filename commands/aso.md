# /aso - Quick ASO Optimization

Generate optimized App Store listing metadata fast.

## Usage

```
/aso [app-name] [description]
```

## Examples

```
/aso
/aso TaskFlow
/aso TaskFlow - AI task manager for professionals
```

## What It Does

1. **Recall** - Checks memory for existing listing data
2. **Collect** - Gathers app details (if needed)
3. **Generate** - Creates optimized metadata for Apple/Google
4. **Validate** - Verifies character limits and no duplicates
5. **Iterate** - Allows refinements until perfect
6. **Save** - Stores approved listing to memory

## Output

Copy-paste ready metadata:

### Apple App Store
- Title (30 chars)
- Subtitle (30 chars)
- Keywords (100 chars)
- Promotional Text (170 chars)
- Description (4000 chars)

### Google Play Store
- Title (50 chars)
- Short Description (80 chars)
- Full Description (4000 chars)

## Time

2-5 minutes

## When to Use

- Need metadata quickly
- Already know your keywords
- Simple app, clear value prop
- Updating existing listing

## When NOT to Use

Use `/aso-audit` instead if you need:
- Competitor analysis
- Keyword research
- Launch planning
- Comprehensive strategy
