## Google Play Specifics

### No Keyword Field
- Keywords are extracted from title and description
- Front-load important keywords in first 2-3 sentences
- Use keywords naturally throughout description

### Formatting
- Use ★ for section headers
- Use ✓ or • for bullet points
- Use line breaks for readability
- Bold text with **asterisks** may work

### Search Indexing
- Title has highest weight
- Short description has high weight
- First 2-3 sentences of full description matter most
- Repeat important keywords 2-3 times naturally

## Validation Checklist

- [ ] Title ≤ 50 characters
- [ ] Short description ≤ 80 characters
- [ ] Primary keywords in title
- [ ] Keywords in first 2-3 sentences of description
- [ ] Clear formatting with sections
- [ ] Contact information included
- [ ] No excessive keyword repetition (stuffing)
- [ ] Natural, readable language
